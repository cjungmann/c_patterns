# sarray Module

This is kind of a self-indulgent tool.  There were three objectives
in taking the time to make this:

1. I wanted to use function pointers to create a flexible and
   efficient algorithm.
2. I wanted to avoid tons of tiny malloc/free pairs that could
   theoretically lead to fragmented memory.  I know, it's not
   likely in this case, but I did say it is self-indulgent.
3. I wanted to use a smaller project to experiment with Doxygen
   styled commenting and documentation.

I'm not going to spend a lot of time talking about this little
project here.  Part of the objective in the Doxygen goal is that
the source code should be self-documenting, especially when using
Doxygen to create the documentation.  Try it and see what you think.
